# 原则模板

只收录会稳定影响判断的原则。每条原则尽量带来源和原话。

| 原则 | 原话或高保真转述 | 适用场景 | 强度 | 备注 |
|---|---|---|---|---|
| 示例：先看是否可持续 | “如果这件事不能长期成立，就不要做。” | 战略、投资、产品 | 高 | 反复出现 |
|  |  |  |  |  |

## 使用说明

- `强度=高`：多次出现，且在真实决策里被执行过
- `强度=中`：只出现过少数几次，但与总体风格一致
- `强度=低`：有表达，但尚未看到稳定实践

优先保留这三类原则：

1. 判断标准
2. 排序标准
3. 取舍标准

少保留这三类材料：

1. 一次性情绪表达
2. 只针对某个偶发事件的评论
3. 没有进入真实决策的漂亮话
