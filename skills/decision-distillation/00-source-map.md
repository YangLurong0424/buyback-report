## 2. 资料来源清单

按可信度从高到低填写，优先一手来源。

| 来源 | 类型 | 时间范围 | 可信度 | 备注 |
|---|---|---|---|---|
| 访谈/演讲/播客 | 一手 |  | 高 |  |
| 书籍/专栏/公开发言 | 一手或近一手 |  | 高 |  |
| 真实案例复盘 | 近一手 |  | 高 |  |
| 团队记录/聊天记录 | 内部材料 |  | 中 |  |
| 二手解读文章 | 二手 |  | 中/低 | 仅辅助，不作硬依据 |

## 4. 喂资料建议

优先顺序：

1. 先填原则和红线
2. 再填决策流程
3. 再补正反案例
4. 最后才补表达风格

如果资料很杂，先问自己三个问题：

1. 这段材料有没有体现“他如何判断”？
2. 这段材料有没有体现“他绝对不做什么”？
3. 这段材料有没有体现“他遇到冲突时怎么取舍”？

只有“能改变判断”的材料，优先级才高。
