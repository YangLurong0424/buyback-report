# 红线与不做清单模板

这里写“只要出现就强烈警惕，甚至直接否决”的东西。

| 红线 | 为什么是红线 | 典型信号 | 处理动作 |
|---|---|---|---|
| 示例：说不清价值从哪里来 | 说明理解不够或对象不成立 | 只能讲概念，不能讲机制 | 暂停/否决 |
|  |  |  |  |

## 分类建议

你可以从四类去整理：

1. 认知红线：不懂、不确定、证据不足时不做什么
2. 人品红线：诚信、价值观、协作方式上的硬问题
3. 模式红线：商业模式、组织机制、增长方式上的硬伤
4. 操作红线：借钱、加杠杆、过度承诺、短期投机等

## 最重要的整理方法

如果某人曾经犯过大错，后来明确说“以后绝不再这样”，这种材料优先级非常高。

因为这类红线通常最能决定真正的决策边界。
