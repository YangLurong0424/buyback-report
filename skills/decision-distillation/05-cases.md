# 案例模板

把成功和失败案例都写进来。不要只收成功案例。

## 成功案例

| 案例 | 背景 | 当时怎么判断 | 最关键的依据 | 结果 | 可复用规律 |
|---|---|---|---|---|---|
|  |  |  |  |  |  |

## 失败案例

| 案例 | 当时错在哪里 | 违背了什么原则 | 后来怎么修正 | 形成了什么红线 |
|---|---|---|---|---|
|  |  |  |  |  |

## 使用建议

优先记录这三种信息：

1. 他为什么在别人都反对时还做
2. 他为什么在看起来很诱人时反而不做
3. 他犯错后，规则是如何变化的

案例的价值不只是“证明他说过什么”，更重要的是“证明他真的这么做过”。
